## Idioms 
> List of idioms 

## Install
```
$ npm install --save idioms 
```

## Usage
```javascript
var Idioms = require('idioms')

Idioms.getIdioms(function(idioms){
  console.log(idioms)
})
```

## API
### .getIdioms(callback)
```javascript
var Idioms = require('idioms')

Idioms.getIdioms(function(idioms){
  console.log(idioms)
})
```
